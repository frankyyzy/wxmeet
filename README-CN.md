## 概要
### 简介
启发于When2meet, 别鸽我是个可以让事件参加者去选择他们有空的时间的微信小程序。但是，比起给电脑浏览器设计的When2meet, 别鸽我可以在所有具备微信的智能手机和平板上使用。因为使用的微信小程序，别鸽我的用户可以参加其他微信用户的事件或者创建并邀请其他微信用户到自己的事件里。



### 软件评估
- **类型:** 工具, 生活
- **简评:** 可以让微信用户和其他微信用户简单方便的计划会见时间。
- **需求:** 需要计划时间的微信用户。
- **客户:** 此软件的主要客户为所有想安排集体会议的微信用户。

- [x] 展示事件时间

### 2. 页面构架

* 简况页面
   * 用户可滑动并看到自己创建的事件。
   * 用户可滑动并看到自己参加的事件。
   * 用户可删除自己创建的事件。
   * 用户可退出自己参加的事件。
   * 用户可上拉刷新事件。
   * 用户可转到创建事件页面去创建事件。
* 创建事件页面
   * 用户可给予事件标题。
   * 用户选择事件有空的日期。
   * 用户可返回简况页面。
   * 用户可确定时间名称和事件日期转到选择时间页面。
* 选择事件页面
   * 用户可滑动选择自己的空闲时间。
   * 用户可滑动取消已选择时间。
   * 用户可返回创建事件页面。
   * 用户可确定以选时间去主事件页面。
* 主事件页面
   * 用户可看到事件其他成员的空闲时间。
   * 用户可按住某个时间点去观看其他在当时间点有空的成员。
   * 用户可返回到简况页面。
   * 用户可修改自己的空余时间。

## 使用帮助 
### 创建事件
1. 在简况页面, 点击"我要组局"。
2. 请给事件添加名称。
3. 选择心仪的日期。
4. 点击"确认提交".
5. 滑动去选择有空的时间。
6. 点击"确认时间"。
7. 事件已被创建且在简况页面展现。

<p align="center">
	<img src = "https://i.imgur.com/zg97UHs.gif" height = 790>
</p>

### 删除事件
1. 在简况页面，点击并按住想删除的事件。
2. 在弹窗上，点击"确定“。
3. 事件已在系统及简况页面中删除。

<p align="center">
	<img src = "https://i.imgur.com/fmdC9lX.gif" height = 790>
</p>

### 分享事件
1. 在简况页面，点击想分享的事件。
2. 在事件里，点击右上角三个点的图标。
3. 点击“转发”。
4. 选择想转发的好友。
5. 被转发的好友可从分享卡片参加此事件。

<p align="center">
	<img src = "https://i.imgur.com/AFrkWn6.gif" height=790>
</p>

### 修改事件
1. 在简况页面，选择想修改的事件。
2. 点击右下角的"修改"。
3. 滑动去调整事件。
4. 点击"确认时间"。
5. 时间已被修改且事件已被更新。

	<img src = "https://i.imgur.com/EOd9zfE.gif" height=790>
</p>
